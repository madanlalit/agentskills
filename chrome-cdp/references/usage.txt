USAGE EXAMPLE: Extracting Hacker News

# 1. Start by listing available targets
$ python3 scripts/cdp.py list
[Target: 9A3B2] https://news.ycombinator.com/

# 2. Navigate and wait for content
$ python3 scripts/cdp.py nav 9A3B2 https://news.ycombinator.com
Navigated to https://news.ycombinator.com
$ python3 scripts/cdp.py wait 9A3B2 .athing
Element .athing found.

# 3. Extract the top 3 story titles
$ python3 scripts/cdp.py eval 9A3B2 "Array.from(document.querySelectorAll('.titleline > a')).slice(0, 3).map(a => a.innerText)"
[
  "Windows 9x Subsystem for Linux",
  "GitHub CLI now collects pseudoanonymous telemetry",
  "3.4M Solar Panels"
]

# 4. Extract points for those top 3 stories
$ python3 scripts/cdp.py eval 9A3B2 "Array.from(document.querySelectorAll('.score')).slice(0, 3).map(s => s.innerText)"
[
  "108 points",
  "105 points",
  "66 points"
]

# 5. Save a screenshot to disk
$ python3 scripts/cdp.py shot 9A3B2 hn_frontpage.png --full
Screenshot saved to hn_frontpage.png
